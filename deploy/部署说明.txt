# 部署到 GitHub Pages（永久有效）

## 操作步骤（纯网页，3分钟）

### 1. 创建仓库
打开 https://github.com/new
- Repository name: `uav-qr` （随便取）
- 选择 **Public**（公开）
- **不要**勾选任何初始化选项
- 点击 **Create repository**

### 2. 上传文件
在仓库页面点击 **uploading an existing file** 链接
- 把 `deploy/` 文件夹里的两个文件拖进去：
  - `index.html`
  - `beep.wav`
- 点击 **Commit changes**

### 3. 启用 Pages
- 仓库页面 → **Settings** → 左侧 **Pages**
- Branch 选 `main`，点 **Save**
- 等待 1 分钟，页面会显示：
  > ✓ Your site is live at `https://你的用户名.github.io/uav-qr/`
